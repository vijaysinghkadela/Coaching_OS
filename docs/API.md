# API Routes Reference

The app is showcase-only, so the routes are demo-safe and do not depend on live third-party services.

## Routes

- `GET /api/auth/callback` - redirects to the requested page
- `POST /api/attendance/mark` - demo attendance response
- `POST /api/attendance/qr-generate` - demo QR token response
- `GET /api/attendance/qr-verify` - demo verification page
- `POST /api/fees/razorpay-order` - placeholder order response
- `GET /api/fees/receipt/[id]` - demo receipt output
- `POST /api/communication/send` - demo broadcast response
- `POST /api/ai/parent-report` - demo report text
- `POST /api/ai/study-plan` - demo study plan text
- `POST /api/webhooks/razorpay` - placeholder webhook route
- `POST /api/webhooks/whatsapp` - placeholder webhook route
