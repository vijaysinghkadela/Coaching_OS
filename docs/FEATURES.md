# Features

Coaching OS is now a showcase-only product with realistic mock data.

## Included Screens

- Dashboard
- Students
- Batches
- Attendance
- Fees
- Tests
- Communication
- Documents
- Staff
- AI demos
- Settings and onboarding

## Data Source

- All data comes from `lib/demo/data.ts`
- `lib/demo/client.ts` mimics the old query layer
- No live Supabase or external backend is required for the UI to work
